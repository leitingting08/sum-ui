# @sum-ui/upload

## Install

```bash
npm i @sum-ui/upload
# or
yarn add @sum-ui/upload
```

## Usage

```vue
<template>
 <sum-upload/>
</template>

<script lang="ts">
import { defineComponent, toRefs } from 'vue'
import SumUpload from '@sum-ui/upload'

export default defineComponent({
  components: { SumUpload },
  setup() {}
})
</script>
```
